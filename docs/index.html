Explore features Coming soon

Problem

Visitors lose signal when they need help most.

Public transport navigation can break down inside stations, tunnels, and crowded tourist areas. NAV-AIDE is designed for those low-signal moments, keeping key travel guidance available locally on the device.

Solution

Local maps, local routing, and on-device assistance.

The MVP is planned around offline assets, local station and POI resolution, OS speech input and output, and on-device LLM inference. The goal is to help users search, route, and recover when they are unsure where to go next.
